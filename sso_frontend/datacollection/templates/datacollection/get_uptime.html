Getting uptime

uptime tells when you last rebooted your device. Obtaining it might be a bit difficult. If you don't want to hop through multiple steps, just scrap it - uptime isn't that important for this study.

Windows

Well, how technical person you are? This requires a few steps. Open cmd.exe (press windows key+R and write cmd.exe). After terminal window opens, run systeminfo | find /i "boot time".

If you don't want to open terminal, open task manager (ctrl+alt+del), go to "Performance" tab and click "Resource monitor" button. Under "Performance" tab (again), there's field "Up time" at the bottom of the page.

OS X

Go to "Applications > Utilities" and click "Terminal" (or use Spotlight and search for "Terminal".). Run command uptime. Output should be something like

12:10 up 3 days, 1:12, 12 users, load averages: 2.07 2.00 1.89

You can copy and paste this whole line (or just relevant parts, "up 3 days, 1:12").

Linux

Open terminal and run command uptime. Copy and paste the output.

Android

If you want to, you can install this small widget. It's free and doesn't require any permissions to run.

iOS

If you want to, you can install this small app. It's quite useful, and free. For uptime, look for "System uptime" field.
